This is the GitHub repo for contest 2 - Comp Arch. This team includes Angelina Martis and Arya Patel.

# How to compile:<br>
To run our project, you can download Eggstimate.asm<br>
OR<br>
Copy-paste the code into an .asm file in the project folder we use in class<br>
Then run without debugging

# "Virus"
If you happen to get a pop-up while debugging saying it contains a "virus"<br>
On Windows, go to settings and then "protection history"<br>
Click on the recent "threat"<br>
Then, click on severe and click restore, and allow your machine to make changes to your device<br>
Then go back and run without debugging
